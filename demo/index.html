zaur-world

Live sky for your location, hour, and weather.

Install 
npm install @nomideusz/zaur-world
import { createWorld } from "@nomideusz/zaur-world";

const canvas = document.querySelector("canvas")!;
const sky = createWorld(canvas, {
  terrain: true,
  satellites: true,
});

sky.destroy(); // on unmount
